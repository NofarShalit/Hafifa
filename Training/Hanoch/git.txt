git
Clone- שכפול git repo קיים מהאינטרנט, לוקח את הsnapshot האחרון
Add- עושה stage לשינויים שביצעת בקובץ, הם אלה שיכללו בcommit
Commit- שמירת השינויים מקומית בcheckpoint חדש בצירוף הערה שאומרת מה נשתנה.
Push- העלאת הbranch שיצרת לפרוייקט אל האינטרנט
Pull- מוריד מידע על branch מהאינטרנט וממזג אותו אל הbranch שלי
Fetch- מוריד מידע על branch מהאינטרנט
Checkout- מעבר לעבודה על branch מסוים
Init- יצירת git repo בתוך תיקיית העבודה בה אני נמצא
Blob- אובייקט המשמש לשמירת קבצים
Tree- אובייקט המשמש לשמירת תיקיות ומצביע על blobים
Commit- אובייקט המשמש לשמירת snapshot של הworking directory- נוצר כל פעם שעושים commit ומצביע לtree
Branch- אובייקט שהוא רצף ישר של commits. כאשר נוצר רצף חדש מסתעף מאחר בפקודה branch נוצר branch חדש.
Tag- אובייקט המצביע על commit ומתייג אותו כגרסה
